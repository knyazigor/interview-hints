## Типы

### Примитивные типы
1. `number`
  - `NaN` тоже `number`
2. `string`
3. `boolean`
4. `null`
  - значение отсутствует или не существует
  - `typeof null === 'object'`
5. `undefined`
  - значение не определено. Пременная объявлена, но ей не присвоено значение
  - `null !== undefined`
6. `bigint`
7. `symbol`


### Ссылочные типы
1. Object
2. Array
3. Set
4. Map
5. WeakMap
